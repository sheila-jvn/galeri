# Mood Board

Gallery app made using React, Mantine and Firebase.

## Run Locally

Clone the project

```bash
  git clone https://github.com/sheila-jvn/galeri.git
```

Go to the project directory

```bash
  cd galeri
```

Install dependencies (**use pnpm**)

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```

## Running Tests

To run tests, run the following command

```bash
  pnpm test
```
